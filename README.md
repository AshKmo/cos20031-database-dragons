# Database Dragons Softdev Major Code

A running instance of this code is available on Mercury at [https://mercury.swin.edu.au/cos20031/s105926680/cos20031-database-dragons/](https://mercury.swin.edu.au/cos20031/s105926680/cos20031-database-dragons/)

Information on the contributions made by each individual to the contents of this repository can be found on the Confluence page "Game Development/Software Major Task Details" under the folder "Major-Specific Tasks Requirements".

The credentials and database can be configured in `credentials.php`. This project requires PHP and a MySQL/MariaDB instance containing the latest version of the SQL implementation of the database, which is available in the Confluence page "Combined test data query" under the folder "SQL Database Implementation".
